# 项目简介

## 核心功能
- 前后端代码与SQL文件
- 完整的万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包

## 适用场景
- 人格测试网站搭建
- 教育和心理咨询服务
- 人力资源评估

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
